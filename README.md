PGPost
======

PGPost is user authentication using the OpenPGP protocol for microblogging and
other online services. This means of authentication is described in `docs/protocol.md`.

Dependencies
------------

System dependencies:
```
    - gpg
    - a MySQL server
```

Python dependencies:
```
    python-gnupg
    python-mysqldb
```
